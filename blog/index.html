Other Places I've Written

Over the years, I've written a few things in a few places.

Quantified Self

From 2012 – 2014, I spent a lot of time exploring the Quantified Self, and wrote up some of my experiments and analyses here. Eventually these explorations led me to work on datafist, which then led to my work on Data Sense with Intel Labs.

Data Sense was, in some sense, the realization of that exploration: I was working on Quantified Self tools full-time! As these things go, that also meant I no longer had time to maintain the blog.
